🔍 VSCode Port Monitor

Real-time Port Status Display in VS Code Status Bar

Version 0.5.1

概要

🎯 開発環境と拡張機能の概要

VSCode Port Monitorは、複数のポートの状態をリアルタイムで監視し、VS Codeのステータスバーに表示する拡張機能です。

開発中のサービスやデータベースの稼働状態を一目で確認でき、プロセス管理機能も備えています。

💻 VS Code 1.60.0+ 🟢 Node.js 16.0+ 🌐 Multi-host Support 🎨 Customizable Display

Port Monitorは、開発者の生産性を向上させるVS Code拡張機能です。 複数のサービスのポート状態を常に監視し、ステータスバーに分かりやすく表示します。

localhost: 300[🟢user:0|⚪️car:1|⚪️2|⚪️3|🟢4]

🔍 マルチポート監視

複数のホストとポートを同時に監視し、リアルタイムで状態を更新

🏷️ インテリジェント設定

ポート範囲の自動展開、Well-knownポート名のサポート

🛑 プロセス管理

ポートを使用中のプロセスを特定し、安全に終了可能

🎨 カスタマイズ可能

アイコン、絵文字、表示形式を自由にカスタマイズ

📖 詳細機能

主な機能

  • 5段階の設定処理: Well-knownポート置換、範囲展開、配列変換など自動処理
  • コンパクト表示: __CONFIG.compact: trueで共通プレフィックスをまとめて表示
  • インタラクティブUI: ステータスバークリックでポートセレクターを表示
  • スマートプロセス検出: サーバープロセスを優先的に識別（Node.js、Python等）
  • 柔軟な設定形式: 配列、オブジェクト、範囲指定など多様な形式をサポート
  • エラー検出: 設定ミスを検出し、具体的な修正方法を提案

設定例

{
  "portMonitor.hosts": {
    "Frontend": {
      "3000": "app",
      "3001": "api",
      "3002": "admin",
      "__CONFIG": {
        "compact": true,
        "separator": "|"
      }
    },
    "Database": {
      "5432": "postgres",
      "6379": "redis"
    }
  },
  "portMonitor.statusIcons": {
    "inUse": "🟢",
    "free": "⚪️"
  }
}

コンパクト表示

共通のプレフィックスを持つポートをコンパクトに表示:

  • 通常: 🟢3000 🟢3001 🟢3002
  • コンパクト: 300[🟢0|🟢1|🟢2]

ポート範囲の展開

{
  "portMonitor.hosts": {
    "Development": {
      "3000-3009": "",  // 10個のポートに自動展開
      "__CONFIG": {
        "compact": true
      }
    }
  }
}

インストール方法

🚀 VS Code Marketplace からインストール

  1. VS Code の拡張機能パネルを開く（Cmd/Ctrl + Shift + X）
  2. "Port Monitor" を検索
  3. インストールボタンをクリック

📦 VSIX ファイルからインストール

# コマンドラインから
code --install-extension vscode-port-monitor-0.5.1.vsix

# または VS Code UI から
1. 拡張機能パネルを開く
2. ... メニューから「VSIXからインストール」を選択
3. VSIXファイルを選択

クイックスタート

1. 基本設定

{
  "portMonitor.hosts": {
    "localhost": [3000, 3001, 3002]
  }
}

2. ラベル付き設定

{
  "portMonitor.hosts": {
    "Services": {
      "3000": "frontend",
      "3001": "backend",
      "5432": "database"
    }
  }
}

3. カスタム絵文字

{
  "portMonitor.portEmojis": {
    "frontend": "⚛️",
    "backend": "🚀",
    "database": "🗄️"
  },
  "portMonitor.emojiMode": "replace"
}
💡 ヒント: ステータスバーの表示をクリックすると、ポートの詳細情報やプロセス管理オプションが表示されます。

プロセス管理

ステータスバーをクリックして、以下の操作が可能です：

ポート選択

監視中のポートから選択し、詳細情報を表示

プロセス詳細

PID、プロセス名、コマンドラインを確認

プロセス終了

確認ダイアログ付きで安全にプロセスを終了

⚠️ 注意: プロセスの終了は慎重に行ってください。重要なサービスを誤って終了させないよう注意が必要です。

設定オプション

利用可能な設定

  • portMonitor.hosts - 監視対象のホストとポート
  • portMonitor.portLabels - ポートのラベル（パターンマッチ対応）
  • portMonitor.portEmojis - カスタム絵文字設定
  • portMonitor.emojiMode - 絵文字表示モード（prefix/replace/suffix）
  • portMonitor.statusIcons - ステータスアイコン
  • portMonitor.intervalMs - 監視間隔（ミリ秒）
  • portMonitor.statusBarPosition - ステータスバー位置（left/right）

Well-known ポート

"http" → 80
"https" → 443
"ssh" → 22
"postgresql" → 5432
"mysql" → 3306
"redis" → 6379
